# Sprint 1 UI – Migliorie incluse
- Tema **dark/light** con toggle e salvataggio preferenza
- 4 **KPI cards** (totale, media mensile, ultimo mese + delta, stanza top)
- Tabella: header **sticky**, righe zebratura soft, numeri a destra
- Grafico invariato con palette tema

## Installazione
Sostituisci i file nel progetto:
- `tailwind.config.ts`
- `src/App.tsx`

Poi esegui:
```
npm run dev
```
e per deploy GitHub Pages l’attuale workflow resta valido.
